# Formulário de Matrícula

## 📋 Descrição

Este é um projeto de formulário de matrícula desenvolvido durante uma aula do curso de Fullstack da Rocketseat. O objetivo é criar uma interface responsiva e acessível para coleta de dados de matrícula, utilizando apenas HTML e CSS.

## 🚀 Funcionalidades

- **Campos de entrada**: Inputs para nome, email, telefone e outros dados pessoais
- **Seleção de opções**: Radio buttons para seleção de gênero, curso, etc.
- **Caixas de seleção**: Checkboxes para seleção de interesses ou confirmações
- **Área de upload**: Drop area para upload de documentos
- **Botões interativos**: Botões estilizados para envio e reset do formulário
- **Design responsivo**: Layout adaptável para diferentes tamanhos de tela
- **Ícones personalizados**: Utilização de ícones SVG para melhorar a experiência visual

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica do formulário
- **CSS3**: Estilização completa, incluindo Flexbox e Grid para layout
- **SVG**: Ícones vetoriais para botões e elementos visuais

## 📁 Estrutura do Projeto

```
Form_Matricula/
├── index.html                 # Arquivo principal do formulário
├── assets/
│   └── icons/                 # Ícones utilizados no projeto
│       ├── buttons/           # Ícones para botões
│       ├── checkbox/          # Ícones para checkboxes
│       └── ilustracao/        # Ícones ilustrativos
└── styles/
    ├── global.css             # Estilos globais (reset, variáveis, etc.)
    ├── layout.css             # Layout geral da página
    ├── index.css              # Estilos específicos da página inicial
    ├── forms.css              # Estilos para formulários
    └── fields/                # Estilos específicos para cada tipo de campo
        ├── buttons.css        # Estilos para botões
        ├── checkbox.css       # Estilos para checkboxes
        ├── droparea.css       # Estilos para área de upload
        ├── index.css          # Estilos gerais dos campos
        ├── input.css          # Estilos para inputs
        └── radio.css          # Estilos para radio buttons
```

## 🎨 Design

O design segue as melhores práticas de UX/UI, com:
- Paleta de cores consistente
- Tipografia legível
- Espaçamento adequado
- Estados visuais para interações (hover, focus, etc.)
- Acessibilidade com contraste adequado e labels descritivos

## 📱 Responsividade

O formulário é totalmente responsivo, adaptando-se a:
- Desktop (largura > 1024px)
- Tablet (768px - 1024px)
- Mobile (< 768px)

## 🚀 Como Executar

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. O formulário estará pronto para uso!

Não há necessidade de servidor ou dependências adicionais, pois é um projeto puramente front-end.
